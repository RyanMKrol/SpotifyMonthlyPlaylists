# BaseNodeProject
A project intended to be forked when starting new Node projects.

# Setup
 - Merge this code into a folder
 - Run `npm install`
 - Change the properties in the package.json:
    - Author
    - Name
    - Description
    - Scripts
    
# Dependencies
- Node v11.14.0
- NPM v6.4.1
- Babel v6.26.0
- Babel preset - es2015
